# Aemeath Mod Localization

This repository contains localization files for the Aemeath character mod for Slay the Spire 2.

## Folders

- `localization/zhs`: Simplified Chinese source text.
- `localization/eng`: English reference text.

## Contributing a Translation

Please add a new language folder under `localization/` and keep the same file names and JSON keys as the existing folders.

For Korean translation work, create:

```text
localization/kor/
```

Then copy the JSON files from `localization/eng/` or `localization/zhs/` into `localization/kor/` and translate only the displayed text values.

See `TRANSLATING.md` for details.
